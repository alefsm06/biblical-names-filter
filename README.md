# Biblical Names Filter

## Descrição
Este é um projeto simples desenvolvido em HTML, CSS (com Bootstrap) e JavaScript para filtrar e exibir nomes bíblicos com base na sua letra inicial. O objetivo principal é criar uma interface amigável, responsiva e de fácil uso.

## Funcionalidades
- Exibição de nomes bíblicos extraídos de dados estruturados.
- Filtro interativo por letras (A-Z) ou exibição de todos os nomes.
- Interface responsiva, projetada para dispositivos móveis e desktops.

## Como usar
1. Faça o download ou clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Utilize os botões na parte superior para filtrar os nomes pela letra inicial ou visualizar todos os nomes.

## Tecnologias utilizadas
- **HTML5**: Para a estrutura do projeto.
- **CSS3** e **Bootstrap**: Para o estilo e a responsividade.
- **JavaScript Vanilla**: Para a lógica de filtragem e manipulação do DOM.

## Estrutura do código
- Os dados dos nomes bíblicos estão embutidos diretamente no arquivo `index.html` como um array JavaScript.
- A lógica de filtragem exibe os nomes em cartões estilizados dinamicamente conforme o filtro selecionado.
